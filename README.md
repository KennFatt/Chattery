# Chattery
A simple multi-threaded chat application over TCP.
![](./.assets/screenshot.png)

## Features
* Configurable
* Multi-threaded
* Absolutely low resource usage

## How to
**Client**  
Currently no official client, but it seamlessly running with `telnet` or `nc` like application.
